Critiquing Shopify Admin Console

Project Summary

In a similar way to how Squarespace allows you to create your own website, Shopify allows you to create your own online store. Users of the service get access to a web based admin console to create and edit their website as well as track analytics, orders, sales channels, suppliers and much more. This presentation will focus on a portion of the platform devoted to the online store.
